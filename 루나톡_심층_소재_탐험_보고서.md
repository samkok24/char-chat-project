## (3) 루나톡 시뮬레이션 작품의 공통 구조 (12개)

### 게임성/시뮬 태그 작품들의 목표·리스크·자원·룰·엔딩 구조

#### 1. **오픈 채팅방 시뮬레이터**
- **예시**: 라벤더톡: 오픈 채팅!
- **구조**: 오픈 채팅방 시뮬레이터 → 다양한 유저 상호작용
- **목표**: 채팅방 운영, 관계 형성
- **자원**: 대화 선택지, 유저 반응
- **룰**: 메신저 시스템
- **프롬프트 응용**: "오픈 채팅방에서 다양한 유저와 소통하는 시뮬"

#### 2. **현대 선협 회귀 계승 시뮬레이션**
- **예시**: 현대 선협 : 무한 회귀
- **구조**: 현대 선협 → 회귀 반복 → 계승 수련 → 하드코어
- **목표**: 수련 완성, 회귀 탈출
- **리스크**: 무한 반복, 실패 누적
- **자원**: 선협 지식, 계승
- **룰**: 회귀 메커니즘
- **프롬프트 응용**: "회귀 반복하며 X를 계승하는 하드코어 시뮬"

#### 3. **5:5 AOS 대전 로그라이크**
- **예시**: 차원 전쟁 : 아포칼립스 로그라이크
- **구조**: 5:5 aos 팀전 → 로그라이크 반복 → 차원 전쟁
- **목표**: 팀전 승리, 로그라이크 클리어
- **리스크**: 팀 패배, 로그라이크 실패
- **자원**: 캐릭터 능력, 팀워크
- **룰**: aos 게임 룰, 로그라이크 순환
- **프롬프트 응용**: "N:N 대전에 로그라이크 요소를 결합한 시뮬"

#### 4. **인터넷 커뮤니티 활동 시뮬**
- **예시**: 인터넷 커뮤니티 시뮬레이션2
- **구조**: 다양한 커뮤니티 → 글 작성/댓글 → 반응 관찰
- **목표**: 커뮤니티 인플루언서, 논란 회피
- **리스크**: 악플, 차단, 여론 악화
- **자원**: 글쓰기 실력, 팔로워
- **룰**: 커뮤니티 규칙, 여론 역학
- **프롬프트 응용**: "다양한 온라인 커뮤니티에서 활동하는 시뮬"

#### 5. **E스포츠 매니저+선수 육성**

## (4) 루나톡 RP/로맨스 작품의 공통 관계 훅 (12개)

### 순애·구원·츤데레·소꿉친구·동거·재회·배신·의심 중심 분석

#### 1. **순애+재회+공포**
- **예시**: 안녕, 내 소꿉친구야. (분명 죽었을 친구가 돌아왔다)
- **구조**: 죽은 친구 → 돌아옴 → 공포+의심+순애
- **감정선**: 슬픔→충격→공포→확인→애정 재점화
- **프롬프트 응용**: "죽었던 X가 돌아와 관계 재구축+공포 요소"

#### 2. **츤데레+의붓 관계**
- **예시**: 하나미 유즈 (츤데레 의붓 여동생)
- **구조**: 의붓 여동생 → 츤데레 태도 → 순애+구원
- **감정선**: 경계→츤츤→데레데레
- **프롬프트 응용**: "츤데레 X와의 의붓 관계에서 순애+구원"

#### 3. **순애+구원+빌런**
- **예시**: S.A.V.E Ver+S ("함께해줘서 고마웠어")
- **구조**: 빌런 → 나와 만남 → 구원 시도 → 순애
- **감정선**: 적대→이해→구원→감사
- **프롬프트 응용**: "빌런 X를 구원하는 순애 관계"

#### 4. **피폐+순애+집착**
- **예시**: 내가 너란 꽃을 주워도 되는 걸까? (피폐 순애)
- **구조**: 피폐한 상대 → 주워줌 → 집착+순애
- **감정선**: 동정→보호→집착→순애
- **프롬프트 응용**: "피폐한 X를 주워 집착+순애로 발전"

#### 5. **순애+동거+공포증**

- **구조**: 공포증 다수 보유 → 동거 → 순애+치유
- **감정선**: 불안→의지→치유→애정
- **프롬프트 응용**: "공포증/트라우마 가진 X와 동거하며 치유"

#### 6. **순애+유명인+비밀 사랑**
- **예시**: 예화 (솔로가수, 당신을 남몰래? 사랑합니다)
- **구조**: 유명인 → 은인에게 비밀 애정 → 순애
- **감정선**: 감사→짝사랑→고백
- **프롬프트 응용**: "유명인 X가 은인에게 남몰래 품은 순애"

#### 7. **하렘+로맨스+선택 갈등**
- **예시**: 카르데아 아카데미! (아카데미 하렘물)
- **구조**: 아카데미 → 다수 여성 → 하렘 관계
- **감정선**: 매력 발산→다중 호감→선택 압박
- **프롬프트 응용**: "X 배경에서 다수와 하렘 관계 형성"

#### 8. **순애+힐링+일상 반복**

- **구조**: 퇴근 → 아내 반김 → 일상 힐링 순애
- **감정선**: 피로→위로→힐링→애정
- **프롬프트 응용**: "일상 반복 속 X와의 힐링 순애"

- **구조**: 도미넌트 여성들 → 복종 요구 → 하렘
- **감정선**: 복종→굴복→애증
- **프롬프트 응용**: "도미넌트 X들의 복종 요구 하렘 관계"

#### 10. **내 앞에서만 X→Y 차별 설정**
- **예시**: 유다현 (내 앞에서만 무표정모드가 꺼지는 후배)
- **구조**: 타인에게 무표정 → 나에게만 감정 표출
- **감정선**: 특별함→호기심→애정
- **프롬프트 응용**: "모두에게 X, 나한테만 Y인 차별 순애"

#### 11. **나만/둘 다 나만 차별 하렘**

- **프롬프트 응용**: "복수 인물이 나한테만 특별한 순애 표현"

#### 12. **구원+빌런+폭력 정당화**
